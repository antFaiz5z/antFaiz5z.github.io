## 数据结构与对象

Redis 的每个键值对都是由对象组成, 键总是一个字符串对象, 值可以是字符串对象、列表对象、哈希对象、集合对象、有序集合对象.

Redis 使用 SDS 用作默认字符串表示, C 字符串只会作为字符串字面量用在一些无需对字符串进行修改的地方如打印日志.

链表是列表键的底层实现之一.

Redis 的数据库使用字典来作为底层实现, 字典还是哈希键的底层实现之一.
字典 使用哈希表作为底层实现.
哈希表 使用链地址法 (separate chaining) 来解决键冲突.
